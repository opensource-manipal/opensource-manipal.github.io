Install streamlit

python -m streamlit_venv .streamlit_venv
source .streamlit_venv/bin/activate
(streamlit_venv) python -m pip install streamlit

Create your streamlit application or download codes pertaining to the book catalogue or sorting algorithms correspondingly from

  • https://github.com/everythingpython/experiments/blob/main/streamlit/03_book_catalogue.py

  • https://github.com/everythingpython/experiments/blob/main/streamlit/07_sorting.py

Run it using -

streamlit run 03_book_catalogue.py

And navigate to the first URL.
